🌾 KrishiSetu

KrishiSetu is a collection of agriculture-based projects designed to help farmers with crop monitoring, recommendations, and smart farming solutions using technology.

🚀 Projects Included

🌱 1. Crop Performance Report 

Uses satellite data (Google Earth Engine)
Calculates NDVI (crop health)
Generates reports
Sends alerts (SMS integration ready)

🌾 2. Crop Recommendation System

Suggests suitable crops based on soil and weather conditions
Helps farmers make better planting decisions

🧪 3. Fertilizer Recommendation

Recommends fertilizers based on soil nutrients
Improves crop yield and soil health

🤖 4. Frontend + Chatbot

Includes chatbot for farmer assistance
Provides quick answers and guidance

🏛️ 5. Government Scheme Updates

Displays latest government schemes for farmers
Helps farmers stay informed about benefits

🌿 6. Plant Disease Solution

Detects plant diseases
Suggests treatments and precautions

🌦️ 7. Weather Prediction

Provides weather forecasts
Helps farmers plan activities accordingly

🛠️ Technologies Used

Python

Machine Learning

Google Earth Engine

HTML, CSS, JavaScript

APIs (Weather, etc.)

NDVI
